# Python Variables – Practice Problems

This file contains practice problems to understand Python variables and assignments.

---

## Problem 1: Store and Print Variables

### Task

Create variables to store:

* your name
* your age
* your city

Print them in one sentence.

### Example Output

My name is Harsha. I am 20 years old and live in Hyderabad.

---

## Problem 2: Sum of Two Numbers

### Task

Create two variables `a` and `b`.
Store numbers and print their sum.

### Example

```python
a = 10
b = 20

sum = a + b
print(sum)
```

Output:
30

---

## Problem 3: Swap Two Variables

### Task

Swap the values of two variables.

### Example

```python
a = 5
b = 10

a, b = b, a

print("a =", a)
print("b =", b)
```

Output
a = 10
b = 5

## Problem 4: Area of Rectangle

### Task

Use variables to calculate the area.

Formula:
Area = length × width

### Example

```python
length = 8
width = 4

area = length * width
print("Area:", area)
```

Output
Area: 32

## Problem 5: Convert Celsius to Fahrenheit

### Task

Convert temperature from Celsius to Fahrenheit.

Formula:
F = (C × 9/5) + 32

### Example

```python
celsius = 25

fahrenheit = (celsius * 9/5) + 32

print(fahrenheit)
```

Output
77.0

## Problem 6: Multiple Variable Assignment

### Task

Assign values to multiple variables in one line.

### Example

```python
x, y, z = 5, 10, 15

print(x)
print(y)
print(z)
```

Output
5
10
15

## Problem 7: Calculate Average

### Task

Store three numbers in variables and calculate their average.

### Example

```python
a = 10
b = 20
c = 30

average = (a + b + c) / 3

print("Average:", average)
```

Output
Average: 20
