# Arrays in Data Structures

## What is an Array?

An array is a data structure used to store multiple values of the same type in a single variable.

Arrays allow fast access to elements using an index.

Example:

```python
arr = [10, 20, 30, 40]

print(arr[0])
print(arr[2])
```

Output

10
30

## Example: Traversing an Array

```python
arr = [1, 2, 3, 4, 5]

for i in arr:
    print(i)
```

Output

1
2
3
4
5
